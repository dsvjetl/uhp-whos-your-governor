# uhp-whos-your-governor

## Recomended requirements
- Node.js (v13.13.0)
- NPM (v6.14.4)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
